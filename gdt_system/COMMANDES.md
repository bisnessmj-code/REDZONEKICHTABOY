# Commandes GDT System

## Commandes Admin

```
/gtstart [mapId]      - Démarrer une partie
/gtstop               - Arrêter la partie
/gtreset              - Reset complet du système

/gtlist               - Liste des joueurs en GDT
/gteqlist             - Comptage par équipe

/gteq [id] [rouge/bleu]   - Changer l'équipe d'un joueur
/gtkick [id]          - Kick un joueur de la GDT
/gtkickall            - Kick tous les joueurs

/gtmaps               - Liste des maps disponibles
/gtann [texte]        - Envoyer une annonce
/gttest [texte]       - Tester une annonce

/gtped [on/off]       - Activer/désactiver le PED
/gtpedopen [true/false]   - Toggle visibilité PED pour tous

/gtdebug              - Afficher infos debug
/gtperms              - Afficher permissions
```

## Commandes Joueur

```
/gtquit               - Quitter la GDT
/gtleave              - Quitter la GDT (alias)
```

## Exemples d'utilisation

```
/gtstart 1            - Démarre la partie sur la map 1
/gteq 5 rouge         - Met le joueur ID 5 dans l'équipe rouge
/gtkick 12            - Kick le joueur ID 12
/gtann La partie commence dans 30 secondes ! (faire une annonce)
/gtped on             - Active le PED
```
